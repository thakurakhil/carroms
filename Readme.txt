How to Run the code:
--------------------------------------------------------------

->Direct compilation: g++ carrom.cpp -lGL -lGLU -lglut -lm  class.cpp
-> Now type './carrom' (on terminal) to run the executable. {in linux}

(or)

-> The makefile compiles the code. (Type 'make' on the terminal)
-> Now type './carrom' (on terminal) to run the executable. {in linux}



About code:
----------------

1.class.h file contains the declarations of all the classes
2.class.cpp file contains the implimentations of the fuctions declared in class.h
3.carroms.cpp contains the main sourcecode
4.help.txt for instrctions
5.Readme for compiling instructions
6.Makefile is the make file for executions
